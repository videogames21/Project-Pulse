# Overview

The goal of this project is to design, implement, and deploy a
full-stack web application using modern AI coding agents. Students will
work in teams to build a system based on provided requirements and
demonstrate the ability to effectively collaborate with AI tools while
maintaining strong understanding of the system they develop.

# Learning Objectives

By the end of this project, students will:

-   Build a full-stack web application (frontend, backend, database)

-   Use an AI coding agent to assist in development, debugging, and
    design

-   Evaluate and refine AI-generated code

-   Deploy a production-ready application to the cloud

-   Demonstrate understanding of all components of their system

# Team Structure

-   Teams of 3-4 students

-   Assign ownership of use cases, but all members must understand the
    full system

-   Students must contribute to BOTH frontend and backend

-   Collaboration via GitHub is required

# Technical Requirements

Your system must include:

-   Frontend user interface in Vue.js and Vuetify (do not use
    ElementPlus)

-   Backend REST API in Spring Boot 4.X

-   Persistent database in MySQL or PostgreSQL

-   Deployment in Microsoft Azure

You must use technologies covered in this course.

# AI Usage Requirements

Students are required to use an AI coding agent (OpenAI Codex or Claude
Code) throughout the project.

You must:

-   Use AI for code generation, debugging, and design

-   Critically evaluate AI-generated outputs

-   Modify and refine generated code

-   Ensure full understanding of your final implementation

# AI Interaction Log (Required)

Each team must submit **exactly 3--4 high-quality AI interaction
examples**:

For each example, include:

-   Prompt(s) used

-   AI response (summary or snippet)

-   What you accepted or rejected

-   What you changed

# Deployment Requirements

-   Must be publicly accessible (live URL)

-   Must use Microsoft Azure

-   Must use a persistent database (no local-only setups)

# Final Reflection (Required)

Each team must submit a short reflection including:

-   3 things AI did well

-   3 limitations of AI

-   2 issues caused by AI-generated code

-   How those issues were resolved

# Demo Expectations

During the final demo:

-   You must demonstrate a working system

-   You must explain any part of your code when asked

Individual understanding will be assessed.

# Grading Breakdown

-   Functionality: 50%

-   System Design: 15%

-   Deployment: 15%

-   AI Usage & Log: 10%

-   Individual Contribution: 10%

# Academic Integrity

Use of AI is required for this project. However:

-   You must understand all submitted code

-   Blind copying without understanding is not acceptable

-   You are responsible for your system's correctness

Build something you are proud of. This project is designed to simulate
modern software engineering practice with AI-assisted development.
